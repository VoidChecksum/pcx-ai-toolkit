# Templates

Starter Enma scripts for Perception.cx. All follow the [12 guidelines](../.claude/skills/game-cheat-guidelines/SKILL.md).

## Single-file starters

| File | What it shows |
|------|---------------|
| [`hello-world.em`](hello-world.em) | Minimal script — lifecycle, one render routine, text drawing |

## Using these

1. Copy `hello-world.em` into your Perception scripting directory.
2. Build/verify in the Perception IDE (`check_script` / Verify button).
3. Use the upstream docs at `https://docs.perception.cx/perception/enma/overview` to discover
   the real PCX API surface before adding any new calls.

This is a minimal, upstream-clean template. The structure and API calls use only symbols that
appear in the two authoritative PCX documentation roots.
